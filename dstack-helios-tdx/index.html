Dstack ❤️ Helios

Helios light clients replicated in a dstack-alike setting that communicates with a P2P networking protocol over QUIC transport which performs mutual authentication to establish a secure encrypted channel for relaying the secret during replication. The nodes that are replicated then spin up a helios light client and a wrapper API around the client that signs the responses of the verified execution with the TEE attested shared dstack key.
